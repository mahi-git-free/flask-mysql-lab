# Phishing URL Management Portal

A web application to manage and track phishing URLs, domains, IP addresses and verdicts.

## Features
- Add phishing URLs with domain, IP and verdict
- Search by URL or domain name
- Edit and delete records

## Tech Stack
- **Backend:** Python, Flask
- **Database:** MySQL
- **Deployment:** Railway
- **Server:** Gunicorn
